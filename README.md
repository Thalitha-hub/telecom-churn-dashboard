## 🎯 Business Problem

Customer churn is a major challenge for telecom companies.  
Acquiring a new customer can cost **5–10x more than retaining an existing one**.

This project builds an **AI decision-support system** that:

- Predicts churn probability
- Classifies customers by risk level
- Identifies key drivers of churn
- Helps prioritize retention strategies

---

## 🧠 Machine Learning Approach

The model was trained using the **Telco Customer Churn dataset**.

Steps:

1. Data cleaning and preprocessing
2. Feature engineering
3. One-hot encoding of categorical variables
4. Logistic Regression model training
5. Model evaluation
6. Explainability using **SHAP**
